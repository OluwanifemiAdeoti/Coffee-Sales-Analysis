# Coffee-Sales-Analysis
Coffee shop sales and order analysis using Python and Power BI

PROJECT OVERVIEW

This project analyzes a coffee shop dataset to uncover insights into sales performance and customer behavior.
The data was cleaned using Python (pandas) and visualized using Power BI.

TOOLS USED

- Python (pandas)
- Power BI
- Jupyter Notebook

KEY INSIGHTS 

- A few products contribute over 60% of total revenue
- Sales peak during afternoon and evening hours
- Card payments dominate transactions (~97%)

FILES INCLUDED

- Data cleaning notebook
- Cleaned dataset
- Power BI dashboard file
- Project report
- Images of Dashboard
